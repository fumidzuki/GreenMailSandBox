# GreenMailSandbox

## Description

「GreenMail」を使用した簡単な単体試験方法についてのサンプルリポジトリです。

## Requirement

* Java：1.8
* JavaMail：1.6.2
* JUnit：4.12
* GreenMail：1.5.10
* Gradle：5.4.1

## Usage

```sh
$ git httpshttps://github.com/fumidzuki/GreenMailSandBox.git
$ cd GreenMailSandBox
$ ./gradlew test
```
