# javamail-sample

## Description

「JavaMail」を使用しての簡単なメール送信方法と単体試験の方法についてのサンプルリポジトリです。

## Requirement

* Java：1.8
* JavaMail：1.6.2
* JUnit：4.12
* GreenMail：1.5.10

## Usage

```sh
$ git https://github.com/fumidzuki/javamail-sample.git
$ cd javamail-sample
$ ./gradlew test
```
